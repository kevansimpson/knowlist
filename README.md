# KnowList

A local-first markdown note-taking app with optional remote sync.

Notes are plain `.md` files on your machine — you own your data.

Some content is generated by AI; accuracy is not guaranteed.

## Getting started

```bash
npm install
npm run dev
```

## Build for distribution

```bash
npm run dist:mac
npm run dist:win
npm run dist:linux
```

## Project structure

```
src/
  main/       Electron main process (file system, git, IPC handlers)
  preload/    Context bridge — exposes window.api to the renderer
  renderer/   React UI
  shared/     Types and IPC channel constants shared across all processes
docs/         Architecture and decision log (published to GitHub wiki)
```
